# NGS Quality Control Pipeline

Mini bioinformatics portfolio project.

## Objective

Exploratory analysis of sequencing reads using Python.

## Input

Public FASTQ sample.

## Workflow

1. Read FASTQ
2. Parse read sequences
3. Calculate read lengths
4. Generate histogram
5. Export QC summary

## Tools

- Python
- pandas
- matplotlib

## Outputs

- read_lengths.png
- qc_summary.csv
